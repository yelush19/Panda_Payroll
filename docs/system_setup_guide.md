## ליתאי ניהול שירותים | פנדה טק

---

## 🎯 סקירה כללית

מערכת web-based לבדיקה אוטומטית של תלושי שכר מול נתוני נוכחות ממקאנו.
המערכת פועלת בדפדפן ללא צורך בהתקנה או שרת.

---

## 📁 מבנה הקבצים

### הקובץ הראשי להפעלה:
```
docs/index.html
```

### קבצי תמיכה נדרשים:
```
docs/
├── index.html                              (🟢 קובץ ראשי)
├── assets/
│   ├── css/
│   │   └── payroll-dashboard-styles.css    (🎨 עיצוב)
│   └── js/
│       ├── payroll-dashboard-ui.js         (🖱️ ממשק משתמש)
│       └── payroll-logic.js                (🧠 לוגיקה עסקית)
└── images/                                 (🖼️ לוגואים)
    ├── logo-litay.png
    └── logo-pandatech.png
```

## 🚀 שיטות הפעלה

### **שיטה 1: הפעלה ישירה (הכי פשוטה)**
1. **פתח את קובץ המערכת:**
   ```
   📂 נווט לתיקייה: docs/
   🖱️ לחץ כפול על: index.html
   ```

2. **בחר דפדפן מומלץ:**
   - ✅ Google Chrome (מומלץ)
   - ✅ Microsoft Edge
   - ✅ Firefox
   - ❌ Internet Explorer (לא נתמך)

### **שיטה 2: GitHub Pages (אונליין)**
אם המערכת מפורסמת באינטרנט:
```
🌐 גש לכתובת: https://[שם-המשתמש].github.io/[שם-הפרויקט]/
```

### **שיטה 3: שרת מקומי (מתקדמים)**
לשימוש עם קבצים מרוחקים:
```bash
cd docs
python -m http.server 8000
```
ואז גש ל: `http://localhost:8000`

## 📋 דרישות מערכת

### **דפדפן:**
- Google Chrome 90+ (מומלץ)
- Microsoft Edge 90+
- Firefox 88+
- Safari 14+ (macOS)

### **רזולוציה מינימלית:**
- 1024x768 פיקסלים
- מסך רחב מועדף: 1920x1080

### **חיבור אינטרנט:**
- נדרש לטעינת ספריות חיצוניות
- לא נדרש לעיבוד קבצים (מקומי)

## 🔧 פתרון תקלות נפוצות

### **המערכת לא נטענת:**
1. ✅ ודא שכל הקבצים במקום הנכון
2. ✅ בדק שהדפדפן מעודכן
3. ✅ נסה דפדפן אחר
4. ✅ בדק חיבור לאינטרנט

### **שגיאות JavaScript:**
```
🛠️ פתח Developer Tools (F12)
👀 בדק שגיאות בקונסול
🔄 רענן את הדף (Ctrl+F5)
```

### **בעיות טעינת קבצים:**
- ודא שקבצי Excel תקינים
- גודל קובץ מקסימלי: 10MB
- פורמטים נתמכים: .xlsx, .xls, .csv

### **בעיות ביצועים:**
- סגור טאבים אחרים בדפדפן
- עבור עם פחות מ-500 עובדים בו-זמנית
- נקה cache הדפדפן

## 🎮 הדרכת שימוש מהירה

### **שלב 1: פתיחת המערכת**
```
📂 פתח: docs/index.html
⏳ חכה לטעינה מלאה (5-10 שניות)
✅ ודא שהלוגואים מוצגים
```

### **שלב 2: העלאת קבצים**
```
📁 טאב "העלאת קבצים"
🎯 גרור קבצי Excel לאזור או לחץ לבחירה
✅ ודא סימוני ✅ ירוקים
```

### **שלב 3: הפעלת בדיקות**
```
🚀 לחץ "התחל בדיקה"
⏳ צפה בהתקדמות
📊 עבור לטאב "ניתוח חריגים"
```

### **שלב 4: סקירת תוצאות**
```
👥 סקור כרטיסי עובדים
⚠️ התמקד בחריגות אדומות/צהובות
📋 עבור לטאב "דוח מנהלים"
```

### **שלב 5: ייצוא דוח**
```
📥 לחץ "ייצוא דוח מלא"
💾 שמור קובץ CSV
📧 שלח למנהלים
```

### 7. 📄 דוח חריגים  
**קובץ:** `/exceptions/index.html`

**מטרה:** אימות ביצוע בקשות מדוח `EXTRAREPORTPAYROLL05.2025.pdf`

**מיפוי בקשות חריגים לשדות בתלוש:**

| בקשה בדוח חריגים | שדה בתלוש | סוג בדיקה | דוגמה |
|-------------------|-----------|-----------|---------|
| **קיזוזים וחיובים** |
| קנס/חיוב | ניכויים שונים | סכום מדויק | אפי: קנס 2,000₪ |
| החזר הוצאות | תוספות שונות | סכום מדויק | חיים: נעליים 450₪ |
| תיקון חשבון בנק | פרטי חשבון | עדכון פרטים | נהיר: בנק 31 סניף 134 |
| חיוב חשמל | ניכוי חשמל | סכום חודשי | משה: 910₪ |
| **עדכוני נתונים** |
| עובד חדש | הופעה בתלוש | קיום תלוש | אורי, איילה |
| תיאום מס | אחוז מס | עדכון אחוז | אורי: תיאום חדש |
| עדכון כתובת | כתובת בתלוש | טקסט חדש | נתן: גן יבנה |
| **התאמות שעות וימים** |
| ניצול חופש | ימי חופש | ירידה ביתרה | דניאל: -2 ימים |
| תוספת שעות | שעות לתשלום | תוספת | איילה: +9 שעות |
| **חריגים מיוחדים** |
| עבודה בחג | תשלום 150% | שעות נוספות | 18.5 חופש רבעוני |
| תלוש אחרון | סיום העסקה | רכיבי פרישה | ליאור: עד 22.05 |

**נקודות קריטיות לבדיקה:**
- **אורי אשתר** - חוזה + ביטוח + תיאום מס
- **איילה שירזי** - חוזה + 101 + 9 שעות
- **אפי** - 2,000₪ מתוך 6,000₪ (3 תשלומים)
- **18.5** - חופש רבעוני לכולם (לא מהצבירה)
